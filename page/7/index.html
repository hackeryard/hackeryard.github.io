深夜一更

喜欢村上，是从小说开始的。最开始看到村上的名字，还觉得是中国的愤青，那一句名言出现在一本示范作文的最下面，大概是，他觉得90后这个词太颓废了。我当时就觉得这个人很有趣，名字也很奇怪。

之后知道他是日本非战后文学的作家。从《且听风吟》到《国境以南 太阳以西》，彻底迷上了村上，因为那时候假期有很多时间可以看书，所以就那样看啊看，觉得自己到底是颓废的，到底很孤独，但这不应该是自卑。后来我明白的时候，是看到村上回答一个读者对《世界尽头与冷酷仙境》的感谢和疑问时的回答，诚如他所说，通过阅读小说，来确信自我（这个自我就很深刻了），来不被外界所摆布。

这两年没怎么看他的小说了，《地下2》看了一半，现在无聊会看看他的问答集，这是去年的活动，有幸在简书里面看到了中文版，翻译的也很有趣啊，在此就不贴图了。

另一个喜爱的作家就是周国平了。高中最喜欢的作家，看过一些他的散文，认识尼采、史怀哲、昆德拉都是因为他的文章。现在也会偶尔看他的散文，但明显跟高中相比，感觉不一样了。大概在于我现在正要奋斗，想要出人头地（商业成功），竟然淡忘了人文关怀。另外很可能是我没那样的境界，近期比较矛盾的原因在于，价值观的混乱让我无所适从，我也在怀疑到底应该怎样认识世界，国平的认识也有其局限性。

晚一点，晚安吧！

360公共字体库停止运行

近期博客打开缓慢，但是我已经把google的字体库替换成了fonts.useso.com了，所以又去查了下，知乎上有人说字体库停止提供服务了，进去官网看到了，的确停止了，所以按照那个知友说的，替换回来googleapis.com，重启服务器后博客打开很流畅！（替换位置：\wp-includes\script-loader.php）

看到360的通知提示说换回google的url，本以为谷歌的字体还是加载的缓慢，但是竟然很流畅，说明国内对字体管制不严，还是这只是暂时，仍然会再次被墙？

如果不想这么折腾，换来换去的话，下载本地字体库也是好选择。唯一的不便就是可能需要更新字体库，我的一个方法是，windows环境下载skyFonts字体同步管理工具，再使用Filezilla上传到指定文件夹。

搬东西的下午

讨论了好几个星期，时间跨度从上学期期末到这学期开始，心里一直都在纠结该不该出去住，现在我的决定还是留在宿舍，跟室友一起，度过大学时光。

虽然还是很无奈的，因为说好了的房子，最后由于没有装修，里面连床都没有，其他电器家具更不用说，房主说等到十月份就可以搬进去了，这一个月他准备置办家具电器，但是啊，都一个月了，中途搬东西肯定不好，开学那段时间算空闲一点，今天又看到一堆的实验，还有重课好多（不知以后能不能更加灵活一点，这样很折磨人，怪不得要发展慕课，感慨万千）。

昨天帮隔壁的室友搬东西，大包小包十来个，其中那个装书的袋子起码几十斤，当时搬进他家属区的房子，新室友说，这个包重，我就突发奇想，说，这比十个大脑还要重（其实很多书都没有好好看，BTW，看到宿舍楼那边论斤卖新书，其实只放在家里或者宿舍里面也是很好）。

室友弄了这么久，真的心累，我帮他请到一辆免费的搬运车，他硬是要请我吃饭，但我坚持认为，那只是我们运气好，碰到了一个善良的人。

他说自己想去美国，但是现在还在按照父亲的想法，学习法语，参加了学校的赴法项目，要是他能去美国读CS，祝福他。我觉得他以后还是有救的，不能只活在父母的期望中，当然有一天你能够证明，自己的选择是正确的。

我觉得自己应该多了解一些行业的情况和研究生计划，考虑自己的兴趣，抓紧时间学习专业知识，俱乐部的事情安排好招新和总体的可行规划后，就不再多插手了。

而我那位同学，以后我还回到丹东去玩的，东北在我心中不知为啥竟然成为一个迷人的地方，愿我们能实现自己的愿想。

以后我还会找他跑步的，最近很虚，上次夸我班上体质最好，然后跑脱了，休息几天，找他玩去。

Life’s Struggle.

开机自启动MacType加载美观字体

因为安装MacType的时候，选择F盘，但是这个软件没有自动生成新文件夹的功能，导致直接就把F盘当做软件根目录了。所以我在F盘下面新建了一个Mactype文件夹，把它的文件都剪切到里面来。

以前安装的时候，设置了开机自启动，但是用cmd下“msconfig”->“启动”->“任务管理器”（也可以直接用任务管理器查看）->“启动”查看时，发现自启动里面的MacType图标是默认的exe图标，不是软件本身的Icon图标，也就是调整程序目录后，系统找不到这些文件了。但又没有添加自启动的选项可以调整，网搜后找到一个方法，如下：

找到安装软件的根目录，把软件主程序，比如“Mactype.exe”，设置一个快捷方式，然后把快捷方式添加到C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp（PS：可以直接把这段地址粘贴到文件管理器的地址栏），这可能需要管理员权限，确认即可。

之后再打开任务管理器，发现MacType的图标已经恢复正常了，以后开机就可以立即使用MacType字体了，不用在等待Mactype加载的时间里，面对不舒服的Clear Type字体了，网友推荐的XHei模式还是很给力的，我觉得比Deepin下面的字体还要规范和美观，推荐大家使用。

处暑的一天

昨天刚到学校，忙着修复电脑之后就睡了，今天起来看着熟悉的阳台和外面的风景，一切如故。

在一个地方，就算是家，也会厌倦吧，而且我还有很多事情要做，还有好多书只开了一个头，还有图书馆可以去逛，还有学校餐厅的日常，即将飘落的梧桐秋叶……

呼吸新鲜的空气，傍晚去跑半个小时步，打乒乓，当然，还要处理一些问题，写一些要交的文档，考虑下俱乐部的规划，最重要的，大三要怎么过？

我的确是用笔思考的那类人，不写出来头脑就混乱，并不是说我数学不好，而是关于选择方面，那种因素很多的艰难的抉择。

室友有参加留法项目的，现在正忙于准备法语学习，有参加数模并准备搬出去主攻托福留学美国的，我如果现在准备的话，是不太晚的。

可当下，我只想处理完琐事，尽心把大三专业课学好，足矣。

今天是二十四节气之处暑，即“处，去也，暑气至此而止矣”，希望烦躁也就此而去~

Win8.1与Deepin双系统安装误将MBR丢失的处理方法

为了尝鲜，安装了桌面Ubuntu之称的Deepin，但是由于这个系统进入很慢，而且除了命令行很快速以外，就连鼠标就没有Win下使用得顺手，而且再次启动进入Win时，速度特别慢，所以想要删除Deepin。

我在EasyBCD按照百度经验提示，把Win7/Vista的Boot Loader写入了MBR，当时以为win8.1的BT也可以用win7的，这太天真了！之后由于没有联网的条件，就关机睡觉了。之后开机发现，根本没有机会加载任何一个操作系统。因为手边只有一个Deepin启动盘，所以就重新安装了Deepin，来到学校后，以前的PE优盘由于存在病毒，我就没有使用网友推荐的做法：用Diskgenius来重建MBR（这时由于Win8.1启动项覆盖了以前的linux的，而windows的boot loader没有转接给linux启动的功能，所以deepin就启动不了）。

而这时就想用Deepin盘做一个PE盘，但在Deepin环境下没找到可用的工具。

最后，通过编辑/boot/grub/grub.cfg(事先修改权限，root下用chmod +w grub.cfg)，在可以自定义的区域添加如下代码：

menuentry “Windows 8.1” {

insmod part_msdos

insmod ntfs

set root=’(hd0,msdos1)’

chainloader +1

}

保存后发现Deepin的图形化“设置”里面，出现了Deepin和Windows 8.1两个启动选择，明天重启测试，应该正常。

How does WordPress work？

WP是一款容易上手的博客管理工具，时下流行的CMS (Content Management System)，运行在LAMP的环境下。（PS:它的Logo有点像大众汽车，似乎暗示着它也会很流行很大众？） 从安装过程可以看出（以Centos7为例），先安装Apache，测试默认页面，再安装Mysql，添加专用数据库给WP使用，同时创建WP使用的用户并赋予此用户全部的特权以使用专用数据库（可以在wp-config.php的修改中看出），这样WP运行时就可以使用WP用户访问和修改专用数据库了。安装完Mysql后安装PHP，因为在WP的目录下除了License、Readme还有各种图标图片之外都是php文件，这些php文件是需要在PHP环境下解释执行的。另外我们需要安装一些模块（modules）以保证和加强功能，比如php-gd，里面含有各种图像处理函数，几乎所有的图像处理都要调用GD库函数，系统说明如下：
