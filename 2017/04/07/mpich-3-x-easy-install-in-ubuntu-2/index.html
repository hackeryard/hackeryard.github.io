MPICH 3.X安装简明指南（2）【ubuntu】

本文基于Ubuntu 14.04安装mpich2，亲测有效：

安装：

[c]apt-get install mpich2 libmpich2-dev mpich2-doc[/c]

测试：

[c]which mpicc[/c]

一般返回：

[c] /usr/bin/mpicc[/c]

另外的步骤按照第一篇文章操作即可。

通过此方法安装的mpich版本为3.0.5，并非最新，可以通过以下命令查看：

[c]mpicc -v [/c]
